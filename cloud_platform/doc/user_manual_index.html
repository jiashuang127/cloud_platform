  1. 平台用户操作手册

产品简介

平台用户是平台应用大厅的访问者，可以浏览并使用平台上架的应用；同时，平台用户也可以访问数据集市，浏览数据集市中的数据，根据自己的需要申请使用，管理员审批通过后，可以直接下载Excel格式的数据进行使用。

使用说明

    • 应用大厅

      • 应用大厅汇总展示平台已经上架的应用，点击应用详情可以查看应用的基础信息，点击应用ICON可以快速访问应用。

        如果应用大厅看不到任何应用有可能有两种情况，第一，学校并没有在应用大厅上架任何应用。第二，当前用户不具备查看应用的权限，如想体验应用可以联系管理员把用户加入到相应的角色组，即可浏览体验该角色组下的应用。

    • 用户中心
      • 文本数据开放

        • 此功能用以申请，并下载使用学校的数据。申请数居前，请先前往数据集市查看数据分类及数据样例，然后在本模块申请数据的页面检索找到需要使用的数据并进行申请，申请时可以细化到具体的表和字段，可以选择时间范围（在时间范围内，可以持续不断的从服务器上获取最新的数据），提交后，等待管理员审核。

          管理员对用户的审批通过后（管理员有权限根据实际情况对用户的申请删减字段或变更授权时间），用户可以在列表中提交／下载数据，下载后，可以在本地获得一份Excel格式的数据表格。

      • 消息中心
          • 消息中心汇总显示管理端向前端推送的消息合集，包括审批类消息的结果、公告类的消息、留言回复信息等，除了在用户中心可以进入外，另外一个入口是导航栏的铃铛标记，铃铛的数字提示未读消息的数量。
        • 系统消息
          • 系统消息汇总展示审核类消息的审批结果，点击标为已读后，确认用户已经阅读过此条消息，导航栏的数字相应减少一位。
        • 平台公告
          • 平台公告汇总显示管理员对平台用户群体推送的消息通知。
        • 留言回复
          • 用户在使用过程中有任何问题，都可以通过页面下端的联系我们给管理员留言，管理员回复的留言会在此页面显示。
      • 基本资料
        • 基本资料页面显示用户的基本信息，信息有误可以修改。
      • 修改密码
        • 修改密码页面可以修改用户的密码，如果接受不到邮箱验证码，说明该校并没有配置发送邮箱验证码的邮箱，可以与管理员取得联系，或者由管理员操作将密码修改为其他复杂密码。
    • 数据集市
      • 数据集市提供用户浏览学校开放的可供下载的数据，数据按照业务属性进行了类别的划分，点击到分类中，可以查看到分类包含的表以及五条脱敏的样例数据，选定数据后，可以回到用户中心，通过文本数据开放的模块，对数据进行申请使用。
